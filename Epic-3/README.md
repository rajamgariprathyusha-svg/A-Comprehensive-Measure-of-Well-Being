# Epic 3: Dataset Download and Understanding

## Overview

This Epic focuses on downloading the Human Development Index (HDI) dataset and understanding its structure. The dataset is explored using Python and the Pandas library to gain insights into its features and prepare it for further analysis.

## Objectives

- Download the HDI dataset.
- Load the dataset using Pandas.
- Understand the dataset structure.
- View rows, columns, and data types.
- Check for missing values.
- Perform basic data exploration.

## Files Included

- `human-development-index.csv` – HDI dataset.
- `understanding_dataset.py` – Code to explore and understand the dataset.
- `reading_dataset.py` – Code to load and read the dataset.

## Technologies Used

- Python
- Pandas
- Visual Studio Code (VS Code)

## Outcome

Successfully downloaded, loaded, and explored the dataset, providing a clear understanding of its structure and preparing it for the next stage of the project.
